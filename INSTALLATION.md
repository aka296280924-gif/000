# 收敛预测指标安装指南

## 快速安装步骤

### 1. 下载指标文件
下载 `convergence_indicator_5min.pine` 文件到本地。

### 2. 在TradingView中导入
1. 登录到 [TradingView](https://www.tradingview.com)
2. 打开任意图表
3. 点击底部的"Pine编辑器"标签
4. 清空编辑器中的现有代码
5. 复制并粘贴 `convergence_indicator_5min.pine` 文件的完整内容
6. 点击"添加到图表"按钮

### 3. 设置图表
1. 将图表时间框架设置为 **5分钟**
2. 确保有足够的历史数据（建议至少100根K线）
3. 调整参数以适合您的交易风格

### 4. 配置参数（可选）
在指标设置中调整以下参数：
- **敏感度**: 根据您的风险偏好选择
- **显示选项**: 开启/关闭预测区域和警报
- **收敛阈值**: 调整检测的严格程度

## 使用提示

- 🔥 图标表示强收敛信号
- ⚠️ 图标表示中等收敛信号
- 红色虚线表示预测的向上突破
- 红色虚线表示预测的向下突破
- 右上角表格显示详细分析信息

## 故障排除

### 常见问题
1. **指标不显示**: 检查是否有足够的历史数据
2. **信号太多**: 降低敏感度或提高收敛阈值
3. **信号太少**: 提高敏感度或降低收敛阈值
4. **多周期数据错误**: 确保您有TradingView Pro订阅

### 技术支持
如有问题，请检查：
- TradingView版本是否支持Pine Script v5
- 图表是否设置为5分钟周期
- 是否有足够的市场数据权限
