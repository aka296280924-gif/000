# 使用示例

## 基本使用流程

### 1. 设置图表
```
时间周期: 5分钟
历史数据: 至少100根K线
市场: 任何支持技术分析的市场
```

### 2. 添加指标
1. 在TradingView中打开5分钟图表
2. 点击"指标"按钮
3. 选择"Pine编辑器"
4. 粘贴指标代码
5. 点击"添加到图表"

### 3. 解读信号

#### 收敛信号
- **🔥**: 强收敛信号（评分85+）- 高概率突破
- **⚠️**: 中等收敛信号（评分65-84）- 中等概率突破

#### 信息标签内容
```
📊 强收敛(5分钟)        # 收敛类型
🎯 向上突破(5分钟)      # 预测方向  
📈 评分:87/100         # 收敛评分
⏰ 预计4根K线后突破     # 突破时间预测
```

#### 视觉元素
- **收敛框**: 红色框标记强收敛，橙色框标记中等收敛
- **预测线**: 虚线指向预测的突破方向和价格位置
- **背景色**: 淡红色/橙色背景提示收敛状态

### 4. 交易策略建议

#### 入场时机
1. 等待收敛信号出现（评分≥65）
2. 确认多周期趋势一致（✅ 趋势一致）
3. 等待突破确认后入场

#### 止损设置
- 强收敛信号：收敛框对侧边界
- 中等收敛信号：最近支撑/阻力位

#### 止盈设置
- 目标1：预测价格的1:1风险回报比
- 目标2：预测价格的1:2风险回报比

### 5. 参数调优

#### 保守设置（减少假信号）
```
敏感度: 低
收敛阈值: 0.025%
回看周期: 36
```

#### 激进设置（增加信号数量）  
```
敏感度: 高
收敛阈值: 0.01%  
回看周期: 18
```

### 6. 警报设置

#### TradingView警报
1. 右键点击图表 > 添加警报
2. 选择指标名称
3. 设置警报条件
4. 配置通知方式

#### 推荐警报文本
```
🚨 5分钟收敛信号检测
收敛评分: {{plot_0}}
预测方向: {{plot_1}}  
预计突破: {{plot_2}}分钟后
```

## 最佳实践

### 适用市场条件
✅ 横盘整理市场
✅ 低波动环境
✅ 明确支撑阻力区域

### 不适用条件
❌ 高波动突发事件
❌ 重要新闻发布期间
❌ 市场开盘/收盘时间

### 风险管理
- 永远设置止损
- 控制仓位大小
- 分批建仓
- 避免过度交易
