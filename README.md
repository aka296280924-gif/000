# 收敛预测指标 - 5分钟专版

一个专为5分钟时间周期优化的TradingView Pine Script技术分析指标，用于检测市场收敛模式并预测突破方向。

## 项目文件

- `convergence_indicator_5min.pine` - 主要指标文件
- `README_INDICATOR.md` - 详细功能说明
- `INSTALLATION.md` - 安装指南

## 快速开始

1. 复制 `convergence_indicator_5min.pine` 文件内容
2. 在TradingView Pine编辑器中粘贴
3. 添加到5分钟图表
4. 开始分析收敛模式

查看 [安装指南](INSTALLATION.md) 获取详细步骤。

## 主要特点

- 🎯 专为5分钟周期优化
- 📊 多技术指标融合分析
- ⏰ 突破时间预测
- 🚨 实时警报功能
- 📱 多周期确认
- 🎨 直观的视觉显示

## 技术指标

- ATR波动率分析
- 价格区间收缩检测
- 成交量萎缩分析
- EMA收敛分析
